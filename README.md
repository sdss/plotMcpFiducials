# plotMcpFiducials

[![Test](https://github.com/sdss/plotMcpFiducials/actions/workflows/test.yml/badge.svg)](https://github.com/sdss/plotMcpFiducials/actions/workflows/test.yml)

## Installation

To install for production, use

```console
pip install .
```

For development

```console
poetry install
```
